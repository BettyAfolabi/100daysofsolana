## Challenge Faced
Encountered an error when trying to export the private key: InvalidAccessException: key is not extractable

This occurred because the private key generated by the SDK is non-extractable by default for security reasons.

## Solution
Exported the private key using `pkcs8` format instead of `raw`, then extracted the last 32 bytes to obtain the private key. Combined it with the public key to form the full 64-byte keypair before saving.

## Result
Successfully created and persisted a wallet (`wallet.json`) that can be loaded and reused, with balance checks working correctly.
